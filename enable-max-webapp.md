Макс не разрешает открывать мобильную версию сайта с телефонов, но она прекрасно работает. Чтобы её открыть нужно подменить `user-agent` в браузере.

Подойдёт любой браузер поддерживающий расширения либо предоставляющий подмену `user-agent` из коробки. Популярным выбором будет Edge и Cromite. Firefox позволяет устанавливать расширения но не поддерживает вэб приложения на Android.

Расширение для подмены можно выбрать любое, например: https://chromewebstore.google.com/detail/user-agent-switcher-and-m/bhchdcejhohfmigjafbampogmaanbfkg

После установки расщирения кликнуть на него, открыть настройки, выбрать `Режим настройки` и вставить следующий текст(Взят ua из 148 версии хрома на виндовс):

```
{
"max.ru" : "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/148.0.0.0 Safari/537.36",
"*.max.ru" : "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/148.0.0.0 Safari/537.36"
}
```
Промотайте вкладку с настройками в самый низ и сохраните изменения.

В будущем если это станет массовой проблемой для Макса они могут полностью отключить мобильный дизайн своего сайта, так что способ вероятно не навсегда.
